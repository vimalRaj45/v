System Overview

Spirelia is an end-to-end AI-powered Lead Intelligence CRM that automatically ingests job alerts from Gmail, qualifies leads using Groq AI (Llama 3.3), enriches company data, and presents everything through a secure, premium web dashboard.

Pipeline Completion 8/10 Steps Complete
flowchart LR A[Gmail Alerts] --> B[Apps Script Trigger] B --> C[Sender Filter] C --> D[Quick Fingerprint Dedup] D --> E[Groq AI Extraction] E --> F[Noise Filter + Scoring] F --> G[Google Sheets - 14 Columns] G --> H[Node.js API Server] H --> I[Web Dashboard]

How to Use

Google Apps Script

  1. Open Google Sheet: 1VJtX69Wn4lDryad8L6NkpMylnlys_tPJqYn-b2Oa_aI
  2. Go to Extensions → Apps Script
  3. Paste the full Code.gs content
  4. Set script property: Project Settings → Script Properties → Add
    • Key: GROQ_API_KEY
    • Value: (your Groq API key)
  5. Run initializeCRM() from the CRM-V2 menu to set up sheets
  6. (Optional) Set a time-based trigger for fetchAndQualifyLeads()

Configure Sender Filter

In Code.gs, modify CONFIG.ALLOWED_SENDERS:

// Process ONLY these senders:
ALLOWED_SENDERS: ["vimalraj5207@gmail.com", "muralidharanl@gmail.com"]

// OR process ALL senders (no filter):
ALLOWED_SENDERS: ["*"]

// OR filter by domain:
ALLOWED_SENDERS: ["@tesla.com", "@boeing.com"]
